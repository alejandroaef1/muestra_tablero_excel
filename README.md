Muestra de tablero en excel (Dashboard.xlsm), contiene una macro para carga de archivos, pulsar el botón cargar y seleccionar el archivo salesweekly.csv, los filtros segmentan datos por trimestre y las tarjetas muestran las métricas correspondientes al día de carga. 

La aplicación transforma y limpia automáticamente el archivo cargado, esta limpieza esta programada de acuerdo a la manera en que viene este archivo en bruto.
